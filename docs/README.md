# Documentacao do Study Lab

Esta pasta concentra os documentos de produto, engenharia, seguranca, testes, planejamento e decisoes tecnicas.

A ordem obrigatoria de leitura esta definida em `../HARNESS.md`.

## Indice

- `product/product-context.md`: resumo do produto e escopo inicial.
- `architecture/clean-architecture.md`: arquitetura, SOLID, responsabilidades e estrutura .NET.
- `security/security-by-design.md`: principios e controles de seguranca.
- `testing/tdd-strategy.md`: estrategia de TDD e tipos de teste.
- `process/versioning-and-repository.md`: fluxo de git, remoto, commits e arquivos proibidos.
- `planning/implementation-plan.md`: plano incremental de implementacao.
- `decisions/`: ADRs e decisoes arquiteturais.
