# ADRs - Architectural Decision Records

Use esta pasta para registrar decisoes tecnicas relevantes.

Modelo sugerido:

```text
# ADR-000X - Titulo

## Status

Proposto | Aceito | Substituido

## Contexto

Qual problema ou restricao levou a decisao?

## Decisao

O que foi decidido?

## Consequencias

Quais beneficios, custos e riscos aparecem com a decisao?
```
