Antes de qualquer implementacao, leia obrigatoriamente o [HARNESS](HARNESS.md).

Fundacao inicial do repositorio:

- documentacao obrigatoria de produto, arquitetura, seguranca, testes e versionamento;
- estrutura de pastas para Clean Architecture;
- configuracao inicial para projetos .NET;
- higiene de repositorio para evitar commit de segredos, dados locais e artefatos gerados.
